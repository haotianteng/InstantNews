// API Documentation

REST API for programmatic access. All responses are JSON. Base URL: http://localhost:8000/api

GET /news

Returns latest news items with sentiment scoring.

limit
Number of items (default 50, max 500)
source
Filter by source name (default "all")
sentiment
Filter: bullish, bearish, neutral (default "all")
q
Keyword search in title and summary
from
Filter from date (ISO 8601, e.g. "2026-03-01")
to
Filter to date (ISO 8601, e.g. "2026-03-02")
curl "/api/news?limit=10&sentiment=bullish"
GET /sources

Returns all active feed sources with item counts and status.

curl "/api/sources"
GET /stats

Returns aggregated statistics: total items, by source, by sentiment, average score.

curl "/api/stats"
POST /refresh

Force refresh all feeds immediately. Returns count of new items found.

curl -X POST "/api/refresh"
GET /docs

Returns this API documentation as JSON.

curl "/api/docs"
